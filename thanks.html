Quick start — Windows
1) Unzip it anywhere, then run RAWKeepers.exe.

Quick start — Mac (Apple Silicon, M1 or newer)
1) Unzip, then move RAWKeepers.app to Applications (or run it from anywhere).
2) First launch: macOS will block it (this beta isn't notarized yet). Open System Settings → Privacy & Security, scroll down, click Open Anyway, then confirm.
3) The app then offers a one-time setup of Adobe DNG Converter (free, straight from Adobe) — install it to unlock RAW → DNG conversion and the best HDR merging.
